# AFormats

**AFormats** is a backend web application designed to manage the submission and evaluation of **Formato A** ‚Äî a required document for the formulation of undergraduate thesis projects at the *University of Cauca*. These projects can be either a **Research Project (TIA)** or a **Professional Internship (PPA)**.

The application facilitates the full lifecycle of a Formato A, which includes:

This backend system handles the data management, validation, and workflow logic required for this process.
